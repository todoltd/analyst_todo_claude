# odoo-tech-solution-writer

Плагін ToDo для написання **Технічних рішень (ТР / ТЗ)** під доопрацювання Odoo 19.
7 модульних скілів збирають повний ТР у єдиному форматі ToDo з брифу, транскрипту зустрічі
(Fathom) або задачі в `erp.todo.ltd`.

## Скіли
| Скіл | Секція ТР |
|------|-----------|
| `tr-scaffold` | каркас документа + «Загальна інформація» + вибір формату (старт) |
| `tr-functional-requirements` | «Загальний опис процесу та функціональні вимоги» |
| `tr-usecases-acceptance` | Use Cases + Acceptance Criteria (Given/When/Then) |
| `tr-odoo-tech-design` | «Опис технічної реалізації» (моделі `td.*`, поля, довідники, права) |
| `tr-effort-instruction` | «Обсяг робіт» (оцінка) + «Інструкція» (тест-кейс) |
| `tr-review` | QA-перевірка консистентності готового ТР |
| `tr-registry-update` | запис у реєстр «Перелік технічних рішень» |

## Швидкий старт
Почни з `tr-scaffold` ("створи ТР по задачі 15498"), далі йди за пайплайном.
Повна навігація — у `docs/0_МАЙСТЕР-ІНСТРУКЦІЯ.md`.

## Структура
```
odoo-tech-solution-writer/
├── .claude-plugin/plugin.json
├── docs/                ← майстер-інструкція + глибокі довідники
└── skills/              ← 7 скілів (по одному SKILL.md)
```
