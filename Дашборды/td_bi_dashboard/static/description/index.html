BI Дашборди для Odoo 19

Нативна BI-платформа всередині Odoo

Модуль td_bi_dashboard — повноцінна BI-платформа всередині Odoo 19, функціонально співставна з Google Looker Studio і Microsoft Power BI. Бізнес-користувач без програмування під'єднується до будь-якої моделі Odoo, будує датасет із фільтрами, агрегаціями та обчислюваними полями, об'єднує кілька моделей (бленд) і конструює інтерактивний дашборд.

Ключова відмінність від зовнішніх BI — модуль виконується всередині Odoo і успадковує модель безпеки Odoo автоматично: кожна вибірка даних виконується від імені поточного користувача через ORM, тому ACL, record rules, groups= на полях і мультикомпанійність застосовуються без дублювання.

Можливості

  • Конструктор датасетів із багаторівневим JOIN і блендом моделей
  • Іменовані міри, обчислювані поля, часовий інтелект (YoY/PoP/ковзне вікно)
  • Інтерактивні дашборди: контроли, cross-filter, drill-down
  • Публікація, frozen-посилання, embed, експорт (XLSX/CSV/PDF), підписки за розкладом
  • Безпека за замовчуванням: без sudo(), агрегація у PostgreSQL, деградація замість відмови
