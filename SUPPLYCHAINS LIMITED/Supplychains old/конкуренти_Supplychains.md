## Галузеві паттерни

**Домінуюча система:** У boutique/small-size сегменті (що відповідає розміру Supplychains) відкрито жодна компанія не розкрила ERP. Галузевий стандарт для HK/Asia mid-size forwarders — CargoWise One (WiseTech Global); для українських гравців — 1С + самописні CRM або Bitrix24. FreightAmigo — виняток із власною платформою.

**Функціональний мінімум галузі:** Онлайн freight calculator / quotation tool; клієнтський кабінет з трекінгом; електронна документація (AWB, коносамент, митні декларації); WhatsApp/месенджер комунікація з клієнтами. Всі знайдені конкуренти мають хоча б частину цих інструментів.

**Конкурент на Odoo:** Не знайдено жодного публічно підтвердженого випадку серед досліджених конкурентів. Odoo у freight forwarding — нішевий вибір, але технічно можливий (є Odoo-модулі для logistics: stock, purchase, accounting, project, CRM). Supplychains може стати першим у своїй конкурентній групі — це диференціатор.

## Сигнали для scope Odoo-проекту

- Ключова функція для freight forwarder на Odoo: **Shipment tracking / операційні картки відправок** (аналог CargoWise shipment file) — потрібна кастомізація або використання Project/Helpdesk модуля.
- Клієнтський портал (Odoo Customer Portal) — прямий аналог "кабінету клієнта" у DiFFreight/BSA Global. Мінімальна кастомізація, висока цінність для клієнтів Supplychains.
- Mitне оформлення та документи: Odoo Documents + кастомні шаблони AWB / Bill of Lading / Customs Declaration — критично для галузі.
- Мультивалютність та мульти-юрисдикція (HKD + USD + EUR + UAH): стандартна Odoo-функція, але потребує правильного налаштування для HK + Shanghai branch.
- Страхування вантажів (згадується у всіх конкурентів): потребує або кастомного модуля, або інтеграції з insurance provider API.
- Drone components / електроніка: hazmat / DGR compliance — спеціалізована документація, Odoo out-of-the-box не покриває. Потрібен окремий checklist-модуль або workaround через Project tasks.
