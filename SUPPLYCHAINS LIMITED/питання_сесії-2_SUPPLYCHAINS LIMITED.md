## Додаток для аналітика — шпаргалка з обліку (прочитати перед зустріччю)

> Щоб упевнено вести облікову сесію, навіть якщо бухоблік — не твоя сильна сторона. Це для тебе, не для клієнта.

### Карта однієї угоди (куди лягає кожне питання)
Гроші від клієнта → обрана **облікова модель** (купівля-продаж із націнкою / транзит + комісія) → оплата фабриці й агентам → їхні витрати лягають на **собівартість** відправки → **виручка** визнається (у який момент?) → потрапляє в **контури** (управлінський — для власника / бухгалтерський — для аудитора) → **закриття місяця** → дані аудитору. Майже кожне питання сесії — одна точка на цьому шляху.

### Головна розвилка простими словами (питання 1)
- **Модель А — купівля-продаж із націнкою:** компанія ніби купує і продає дорожче. В оборот (виручку) іде **вся сума угоди**, прибуток — різниця.
- **Модель Б — комісія / транзит:** гроші клієнта це **не виручка**, а транзит (проходять через компанію до фабрики). Виручка — **лише винагорода** компанії; решта — зобов'язання перед фабрикою. Це модель Yodezeen (транзитний рахунок).
- **Як розпізнати:** спитай «чи стоїть цей товар колись на вашому балансі?» і «що йде в оборот для аудитора — вся сума чи тільки ваша винагорода?». Не вгадуй — хай підтвердить бухгалтер.

### Касовий метод vs метод нарахування (питання 20)
- **Касовий:** дохід/витрата визнаються **коли пройшли гроші** (оплатили / отримали).
- **Нарахування:** визнаються **коли є подія/документ** (відвантажили / отримали рахунок), навіть якщо ще не оплачено.
- Часто: управлінський контур ведуть **за оплатою** (касовий), бухгалтерський — **за документом** (нарахування). Якщо клієнт каже «дохід — коли клієнт заплатив», це касовий.

### Контури обліку (питання 11)
- **Бухгалтерський (регламентований):** для держави й аудитора, за правилами Гонконгу.
- **Управлінський:** для власника — реальний прибуток і маржа по відправці.
- **Як реалізуємо:** один план рахунків + аналітика через **Analytic Accounts** (мітки по клієнту/відправці), рахунки **не дробити** (досвід Бріолекс).

### Міні-глосарій
- **Виручка / оборот** — гроші, зароблені від продажів (до вирахування витрат).
- **Собівартість** — скільки коштувало те, що продали (закупівля + доставка / landed cost).
- **Маржа / прибутковість** — виручка мінус собівартість.
- **AP, кредиторська заборгованість** — скільки **ми винні** постачальникам/агентам.
- **AR, дебіторська заборгованість** — скільки винні **нам** клієнти.
- **Перевиставлення (pass-through)** — оплачену нами витрату (напр. фрахт) виставляємо клієнту, інколи з націнкою.
- **Транзитний рахунок** — «тимчасова поличка» для чужих грошей, що проходять через нас.
- **План рахунків (CoA)** — перелік усіх рахунків обліку (каса, банк, доходи, витрати…).
- **Analytic Accounts** — мітки в Odoo для аналітики (клієнт/відправка/менеджер), не змінюючи план рахунків.
- **Проводка** — запис операції в обліку (дебет/кредит).
- **Закриття періоду** — «замкнути» місяць: усе рознесено й звірено, далі не міняємо.
- **Нарахована витрата (accrued)** — витрата вже є, а рахунок ще не прийшов; ставимо очікувану суму.
- **Курсові різниці** — переоцінка валютних залишків за курсом (тут **не ведуться**, X-17).
- **Profits tax (Гонконг)** — податок на прибуток; деталі — питання бухгалтера/аудитора, не обіцяй від себе.
